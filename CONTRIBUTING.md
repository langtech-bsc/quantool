# Contributing to quantool

Thanks for your interest in contributing!

## How to Contribute

- Fork the repository and create your branch from `alpha-release`.
- Follow the coding style used in the project.
- Write clear commit messages.
- Add tests for new features or bug fixes.
- Ensure all tests pass before submitting a pull request.

## Reporting Issues

- Use GitHub Issues to report bugs or request new features.
- Please provide as much detail as possible.

## Pull Request Process

- Describe your changes in the pull request.
- Reference related issues if applicable.
- The team will review and provide feedback.

## Code of Conduct

- Be respectful and constructive in all communications.

For more details, see the [README.md](README.md).
